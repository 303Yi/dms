# niudao
这是一个开源的在线学习知识共享平台

访问地址是：http://www.niudao.cn  

这个域名是我朋友的，他非常支撑搭建整个平台，所以作为支撑和鼓励送的，非常给力。

目前未部署，相关环境准备中。



**一共划分3个版本，分别对应不同的人群 ：**

入门级的是SSM，通过入门级版本的训练，能很好的掌握后端技术，前端技术；

中级版是分布式，踏入微服务的基础，初尝分布式架构的开发方式； 

高级版是分布式+高性能缓存/检索+大数据自动采集和处理，增加大数据检索方面；
